# The CLI Tool

## Configuration

The system can be configured by adding a `.taskdn.config.json` to your home directory. This is read by The only information that **must** be included are three absolute paths:

- `areas_dir` - Path to the directory containing your Area documents.
- `projects_dir` - Path to the directory containing your Project documents.
- `tasks_dir` - Path to the directory containing your Task documents.
