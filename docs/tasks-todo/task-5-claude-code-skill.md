# Phase 5: Claude Code Skill

Claude Code skill and commands for AI-native task management. Skill for working with the system using the CLI.

## Scope

Shipped as a Claude Plugin for Claude Code.

- [ ] Claude Skill
  - [ ] SKILL.md
    - Context about the task system & philosophy
    - How to use the CLI
    - Common examples
    - "When to do what" table.
  - [ ] Resources dir
    - Copy of the spec
    - Full CLI docs
    - Explainer about Template
    - Explainer about wider ecosystem (SDKs, Obsidian etc)
  - [ ] Templates dir
    - [ ] Various templates for new areas, projects and tasks
  - [ ] Toools dir
    - [ ] Bundled CLI tool
- [ ] Custom slash commands for common operations
- [ ] Subagent to appraise whole system in full and return a summary to the main agent.

## Notes

Can potentially be developed in parallel with or before the CLI, as it may use file operations directly or wrap the CLI.
