## Overview

Prepare estimated taxes for Q1 2025. Need to gather all income documentation, calculate estimated payments, and file on time.

## Key Dates

- **Jan 15:** Q4 2024 estimated payment due
- **Apr 15:** Q1 2025 estimated payment due + annual return

## Documents Needed

- [ ] 1099s from clients
- [ ] Bank statements
- [ ] Investment account statements
- [ ] Previous year return for reference
- [ ] Expense receipts and records

## Process

1. Gather all income documentation
2. Compile deductible expenses
3. Calculate estimated tax liability
4. Review with accountant (if needed)
5. File and pay

## Notes

Consider whether to increase estimated payments this year based on income trajectory.
