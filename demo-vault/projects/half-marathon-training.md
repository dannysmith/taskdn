## Goal

Complete a half marathon (21.1km) in under 2 hours.

## Target Race

City Half Marathon - April 20, 2025

## Training Plan

12-week plan starting January 6.

### Weekly Structure
- **Mon:** Rest or cross-training
- **Tue:** Speed work / intervals
- **Wed:** Easy run
- **Thu:** Tempo run
- **Fri:** Rest
- **Sat:** Long run
- **Sun:** Recovery run or rest

### Phase 1: Base Building (Weeks 1-4)
Build up weekly mileage gradually. Focus on easy runs.

### Phase 2: Building (Weeks 5-8)
Introduce tempo runs and longer long runs.

### Phase 3: Peak (Weeks 9-11)
Highest mileage weeks. Include race-pace work.

### Phase 4: Taper (Week 12)
Reduce volume, maintain intensity. Rest before race.
