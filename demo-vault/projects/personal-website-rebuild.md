## Overview

Time to rebuild the personal website from scratch. The current site is outdated and doesn't reflect current work or capabilities.

## Goals

1. Modern, fast, accessible site
2. Easy to update and maintain
3. Showcase recent projects and writing
4. Good SEO and discoverability

## Tech Stack

- **Framework:** Astro
- **Styling:** Tailwind CSS
- **Hosting:** Vercel
- **CMS:** Markdown files (same as this vault!)

- [ ] Home / landing
- [ ] About
- [ ] Projects showcase
- [ ] Blog / writing
- [ ] Contact

## Design Notes

Keep it simple. Focus on content over flashy design. Fast loading times are a priority.
